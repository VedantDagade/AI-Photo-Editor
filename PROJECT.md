
# AI Image Editor-:
 - Start App
  ```
  npx create-next-app@latest
  ```
  - Shadcn ui
   ```
   npx shadcn@latest init
   ```
  - Add components of shadcn like this
   ```
   npx shadcn@latest add button
   ```

   ```
   npm install next-themes
   ```
